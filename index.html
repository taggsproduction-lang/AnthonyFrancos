Implementation

How We Get There

A phased rollout that proves value fast — start small, scale what works

1
Foundation Weeks 1–4
  • Pull customer data export from Prism (no POS changes needed)
  • Set up CRM system, import and segment customers
  • Build 4 automated email flows (win-back, welcome, follow-up, weekly)
  • Audit and optimize Springfield's Google Business Profile
  • Design and print in-store data capture cards (SMS opt-in)
  • Establish baseline repeat customer metrics
2
Activation Weeks 5–10
  • Launch win-back campaign to lapsed customers — results in 7 days
  • Activate welcome series for all new web orders
  • Deploy SMS opt-in system and first text campaigns
  • Begin structured social media content calendar
  • Push review generation — target 10+ new Google reviews/month for Springfield
  • First monthly performance report delivered
3
Optimization & Scale Months 3–6
  • A/B test email subject lines, offers, and send times
  • Launch Facebook/Instagram retargeting ads
  • Refine customer segments based on 90 days of real data
  • Introduce seasonal campaigns (holidays, summer specials)
  • Full performance review at Month 3 — provable results delivered
  • Compile website improvement recommendations for Phase 2
  • Evaluate expanding strategy to Wayne location based on Springfield results
